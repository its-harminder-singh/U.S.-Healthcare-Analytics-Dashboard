🏥 US Healthcare Analytics Dashboard
1. Project Title

US Healthcare Analytics Dashboard | Power BI

2. Project Description & Purpose

This project analyzes healthcare data to uncover patterns in patient demographics, medical conditions, hospital admissions, billing amounts, and insurance providers. The dashboard was built to demonstrate how healthcare organizations can leverage data analytics to improve operational efficiency, optimize resource allocation, and gain actionable insights into patient care and financial performance.

The primary goal of this project is to transform raw healthcare data into meaningful visualizations that support data-driven decision-making.

3. Technologies Used
Power BI – Data Visualization & Dashboard Development
Power Query – Data Cleaning & Transformation
DAX – KPI Calculations and Measures
Microsoft Excel/CSV – Data Source

4. Data Source
The dataset used in this project is a publicly available healthcare dataset obtained from Kaggle for educational and portfolio-building purposes.

Dataset contains approximately 9,976 patient records including:

Patient Demographics
Medical Conditions
Admission Types
Billing Amounts
Insurance Providers
Hospital Information
Medication Details
Test Results

🔍 Key Insights & Findings
1. Hypertension was the most common medical condition
Accounted for over 21% of total patient records, highlighting its significant impact on healthcare demand.
2. Cancer generated the highest average billing amount
Cancer treatments incurred the highest healthcare costs, averaging approximately $39.7K per patient.
3. Emergency admissions represented a substantial share of hospital visits
Indicating the importance of emergency resource planning and staffing.
4. Patient age distribution was concentrated in middle-aged and older populations
Suggesting increased healthcare utilization among aging demographics.
5. Billing amounts varied significantly across medical conditions
Revealing opportunities for cost optimization and resource allocation.
6. Insurance providers showed comparable patient coverage volumes
Indicating a competitive and relatively balanced insurance landscape.
🎯 Business Impact
